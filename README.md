# Destiny 2 Godroll Wishlist for DIM

### Retr0gr4d3 Gods All:
- Contains all ~ 2200 guns. Not every weapon has enough data to make solid judgement, so you probably do not want to use this.
  - [Retr0gr4d3_Gods-all.txt] 2208 weapons, 14738 rolls, 0 skipped
### Retr0gr4d3 Gods:
- Favours 15% or higher popularity and usage per perk.
  - Allows for a broad range of perks which fit into a god/curated roll, like you would find on Light.gg or Godroll.tv
  - [Retr0gr4d3_Gods.txt] 1571 weapons, 14101 rolls, 637 skipped
### Retr0gr4d3 Gods Strict: 
- Favours 50% or higher popularity and usage per perk.
  - Tightens results to just the top half of the godroll perks.
  - [Retr0gr4d3_Gods-strict.txt] 872 weapons, 5647 rolls, 1336 skipped
### Retr0gr4d3 Gods Perfect:
- Favors only the TOP 100% popularity and usage per perk.
  - Heavily restricts all rolls to just their "perfect godroll" status. Accounts for the last 2 perk columns OR all 4 perk columns, if applicable.
  - [Retr0gr4d3_Gods-perfect.txt] 613 weapons, 1402 rolls, 1595 skipped

> [!NOTE]
> `Strict` and `Perfect` reduce the overall perk pool further than the base Retr0gr4d3 Gods. You typically want to use the base file unless you're filtering a large quantity of godrolls.
